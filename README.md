# PVRecyclerViewBaseAdapter

This is a library for help coding with adapter recyclerview.

## Implementation

### JitPack
```
allprojects {
    repositories {
    ...
      maven { url 'https://jitpack.io' }
    }
}
```

### Gradle
```
dependencies {
    implementation 'com.github.VichitGit:PVRecyclerViewBaseAdapter:0.1.0'
}
```
